# 新建文件时快速生成Html

安装如下插件：

  • FileHeader：自动创建文件开头模板，并且会根据最后的保存时间修改更新时间。官网链接 (opens new window)。
  • CSS Format：css格式化。
  • Emmet：它能够让你在编辑器中书写CSS和HTML的缩写并且动态地拓展它，是一个能大幅度提高前端开发效率的一个工具。这个软件的安装过程比较久。官网教程 (opens new window)。

开始使用：

新建文件，输入html:5，按[Ctrl + E] 或者 Tab 键，

参考链接：Sublime Text 新建文件快速生成Html【头部信息】和【代码补全】、【汉化】 (opens new window)

# Sublime Server

我们如果右键在浏览器中打开html网页，其实是以文件路径的方式打开的，导致很多api无法操作。最好的办法是：将html在服务器上打开。

我们如果安装 Sublime Server，就可以让网页在本地的服务器上打开。

安装成功之后，使用步骤如下：

（1）选择菜单栏"Tools --> SublimeServer --> Start SublimeServer"启动服务器。

（2）在html文档里，右键选择 "View in SublimeServer"。

如果想关闭服务器，直接把Sublime Text 整个软件关掉就好。其他的关闭方式容易导致软件卡死。
